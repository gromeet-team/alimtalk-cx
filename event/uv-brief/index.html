UVID 선패드 론칭과 함께 진행하는 UV 카메라 체험형 바이럴 이벤트. 단순 후기 수집이 아니라 고객 체험 강화 · UGC 확보 · 광고소재 재활용 · 재구매 유도를 한 번에 가져가는 구조로 설계합니다.

01 / BACKGROUND

왜 이 이벤트인가 — 배경과 목적

제품의 진짜 매력

UVID 선패드는 가격 비교로 승부하는 일반 선크림이 아니다. 메이크업 위에서도 다시 바를 수 있고, UV 카메라로 차단이 눈에 보이는 새로운 체험을 주는 선케어다.

기존 후기 이벤트의 한계

단순 "후기 써주세요" 이벤트는 이 제품의 와우 모먼트를 담지 못한다. 자외선 차단은 시각적 증거가 있을 때만 확실한 설득력이 생긴다.

네 마리 토끼

① 고객 체험 강화 · ② UGC 확보 · ③ 광고소재 재활용 · ④ 재구매 유도. 이 네 가지를 한 이벤트 구조 안에서 동시에 회수한다.

광고비 대체 수단

현재 원가 구조상 본품·쿠폰·유비드 패키지를 보상으로 사용해도 메타 광고비 대비 ROI가 더 높은 구조다. 성과형 리텐션 비용으로 쓰되 철저히 관리한다.

02 / STRUCTURE

리뷰 · 리텐션 전체 구조

고객 여정의 각 단계에서 필요한 행동을 정확히 유도하도록 보상을 4단계로 분리했다. 각 단계마다 목적이 다르기 때문에 보상도 다르게 설계한다.

01
포토리뷰
첫 후기 진입장벽을 낮추는 용도. 현금성보다 브랜드 결이 맞고 실제 사용성과 연결됨.
휴대용 케이스
02
한달리뷰
장기 사용 후기 확보 + 재구매 유도가 핵심. 선집행 광고비가 아니라 실제 재구매 시점에 쓰는 비용.
15,000원 쿠폰
03
VOC 인터뷰
실제 개선 인사이트 확보. 카피 · 상세 · 광고 메시지에 바로 반영할 수 있는 고밀도 데이터 수집 목적.
본품 1개 추가
04
UV 카메라 체험
단순 후기가 아닌 체험형 바이럴 + 광고소재 확보 이벤트. 참여형 / SNS형 두 트랙으로 운영.
유비드 180일 / 360일 패키지
STEP 01
체험
UV 카메라 대여, 선패드 사용 전후 촬영
STEP 02
UGC 확보
before/after 영상, 사용 후기 수집
STEP 03
재활용
광고 소재, 상세페이지, 메타 광고
STEP 04
재구매
쿠폰 · 패키지 · 팬층 형성
03 / CUSTOMER TOUCHPOINTS

고객은 어떻게 이 이벤트를 알게 되나

기존 내돈내산(+21일) 이벤트는 제거하고, 배송완료 +1일 시점에 UV 이벤트 모집 메시지로 전환한다. (제품 받고 가장 궁금할 때가 골든 타임)

경로 역할 담당
제품 삽지 QR 박스 개봉 후 가장 먼저 접하는 접점. 훅 + 보상 + QR 3요소. 엘렌
배송완료 +1일 알림톡 "리뷰 독촉"이 아니라 "궁금할 때 UV 체험 신청" 유도 톤. 루미 · 조이
상세페이지 이벤트 섹션 구매 전에도 "이런 체험까지 연결된다"는 기대감 형성. 엘렌
전용 랜딩페이지 무슨 이벤트 · 뭘 받는지 · 어떻게 신청 · FAQ 한 번에 이해. 현비스

05 / KEY DECISIONS

확정된 것과 하지 않을 것

확정 (DO)

  • 포토리뷰 = 휴대용 케이스
  • 한달리뷰 = 15,000원 쿠폰 (1인1회 · 중복불가 · 14일)
  • VOC 인터뷰 = 본품 1개 추가
  • UV 이벤트 = 유비드 180일 / 360일 패키지
  • 배송완료 +1일 알림톡 트리거
  • 공개 랜딩 별도 제작, 공식몰(uvid.co.kr) 연결
  • 메타 이벤트: PageView + SubmitApplication
  • 향후 CAPI로 Lead까지 확장 검토

제외 (DON'T)

  • 기존 내돈내산(+21일) 이벤트 삭제
  • 본품 3개 / 6개 보상안 폐기 — 임팩트 약함
  • 그로밋 대시보드에 고객용 붙이기 — 내부용 유지
  • ViewContent / content_ids — 카탈로그 일치율 이슈
  • 카피 · 설명 과다 랜딩 — 모바일 퍼스트로 축소
  • 일반 후기 요청 톤 알림톡 — 체험 신청 유도 톤으로
